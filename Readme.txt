Welcome to PPStartUp Demo version 1.0

To run this demo:

1. Copy the "0 PPStartUp" folder and paste it into "My Documents"

2. Inside the "0 PPStartUp" folder you will find the "PPS Files" folder

3. Inside the "PPS Files" folder, open the excel file named "PPStartUp Invoicing"

4. Enjoy the demo
